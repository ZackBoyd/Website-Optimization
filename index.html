Welcome to my fourth Udacity Frontend Nanodegree project, website optimization.

Steps to run:

  • Go to http://zackboyd.github.io/Website-Optimization
  • Download .zip file
  • Open index.html in browser

Optimization goals:

  • PageSpeed score >90 on mobile and desktop for index.html
  • <5ms time for pizza resizing
  • 60 fps when scrolling

Steps taken to optimize site:

  • Resized and compressed multiple images loaded in pizza.html and index.html
  • Moved style.css and print.css to style tags in index.html
  • Rewrote for loops that resize pizzas and handle scrolling to remove unnecessary DOM queries and forced synchronous layouts
  • Replaced selector queries with getElementById and getElementsByClassName to speed up queries
  • Added async to js scripts
  • Minified css
